## 快速开始

### 新建文章

``` bash
$ hexo new "文章标题"
```

更多说明请参考：[Hexo 写作文档](https://hexo.io/zh-cn/docs/writing)

### 本地预览

``` bash
$ hexo server
```

启动后可访问 `http://localhost:4000/` 查看效果。

### 生成静态文件

``` bash
$ hexo generate
```

该命令会在 `public` 目录生成可部署文件。

### 部署到远程

``` bash
$ hexo deploy
```

更多说明请参考：[Hexo 部署文档](https://hexo.io/zh-cn/docs/one-command-deployment)
